деструктуризация объекта

вложенный объект -- let {name: {ferst, last}} = obj

сохранить под другим именем -- let {name: {ferst: ferstName, last: lastName}} = obj

по умолчанию -- let { last = 'smit' } = obj (не работает для вложенных)

"рэст элемент" все в одну переменную -- let { ...ferst } = obj  (получим обект с полями)
